Projeto 1 - Power BI

O Projeto envolveu a criação de um dashboard no Power BI utilizando a base de dados US Candy Distributor, que continha informações sobre vendas, produtos, fábricas e metas. O objetivo foi desenvolver uma visualização interativa para analisar o desempenho de vendas, comparar metas e avaliar a distribuição geográfica e dados das fábricas, facilitando a tomada de decisões estratégicas.

Projeto 3 - Power BI

O projeto envolveu a criação de um dashboard no Power BI para um e-commerce, com um foco maior na segmentação de clientes e na análise das métricas RFV (Recência, Frequência e Valor). O objetivo foi proporcionar uma visão estratégica do comportamento dos consumidores, identificando os perfis mais valiosos e assim possibilitando otimizar as ações de marketing e retenção.

Projeto 4 - Análise de Dados

Foi realizada uma análise de dados para avaliar o impacto dos investimentos em publicidade online (YouTube, Facebook e jornais) nas vendas. O objetivo foi entender a relação entre os gastos com anúncios e o retorno financeiro. Além disso, a análise incluiu a construção de um modelo preditivo para estimar o retorno de vendas com base nos investimentos, auxiliando na tomada de decisões e na alocação eficiente do orçamento publicitário.

Projeto 5 - Análise de Dados

Foi realizado uma análise de dados para uma empresa de e-commerce o qual o objetivo é entender o comportamento dos clientes para personalizar suas campanhas de marketing. Usando dados de clientes, produtos e transações de 2010 a 2011 foi feito uma clusterização. A partir da clusterização, é possível gerar insights para que a empresa possa segmentar melhor a sua base de clientes e personalizar as suas campanhas de marketing, direcionando promoções e ofertas aos clientes com base no comportamento de compras.
